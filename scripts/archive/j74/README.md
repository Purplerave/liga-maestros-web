# Archivo de Jornada 74

Estos scripts conservan la importacion y las correcciones puntuales usadas para
la Jornada 74. No forman parte del flujo semanal actual y no deben ejecutarse
contra una base de datos de produccion.

Para nuevas jornadas usa las herramientas genericas de la raiz:

- `SCRAPE_QUINIELA15_PROXIMA.py`
- `IMPORTAR_PROGRAMA_JORNADA.py`
- `AUDITAR_JORNADA_LIGA_MAESTROS.py`

El contenido se mantiene solo como referencia historica. Algunas utilidades
antiguas contienen rutas locales o datos parciales de aquella jornada.
